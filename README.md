# Ελεγκτής ταινίας LED

Ο στόχος του project είναι η δημιουργία ενός ελεγκτή ο οποίος να προσομοιώνει το φυσικό φως της ημέρας χρησιμοποιώντας μια ταινία LED (NeoPixel). Το σύστημα ελέγχεται από έναν μικροελεγκτή Raspberry Pi Pico W, διαθέτει αισθητήρα για την μέτρηση της θερμοκρασίας και της υγρασίας και προσφέρει ένα web interface για απομακρυσμένο έλεγχο.

## Λειτουργία ελεγκτή

Η "καρδιά" του ελεγκτή είναι το αρχείο main.py, το οποίο υλοποιεί τις παρακάτω λειτουργίες:

1. Το αρχείο αυτό τρέχει στον μικροελεγκτή και εκτελεί, παράλληλα, κάποιες εργασίες χρησιμοποιώντας τη βιβλιοθήκη uasyncio:

    - Σύνδεση στο WiFi & Συγχρονισμός ώρας: Συνδέεται στο WiFi χρησιμοποιώντας τις ρυθμίσεις από το settings.json. Συγχρονίζει την ώρα του μέσω διαδικτύου (NTP) και την προσαρμόζει στην ώρα Ελλάδος (υπολογίζοντας τη θερινή/χειμερινή ώρα).

    - Λειτουργία LED:
        Auto Mode: Κάθε 50ms ελέγχει την ώρα. Η συνάρτηση get_sun_color() καθορίζει το χρώμα (π.χ. πορτοκαλί για ανατολή, λευκό για μεσημέρι, μπλε-μαύρο για νύχτα).

        Manual Mode: Τα LED παίρνουν το χρώμα και τη φωτεινότητα που επιλέγει ο χρήστης από το κινητό ή τον υπολογιστή του.

    - Αισθητήρας: Κάθε 30 δευτερόλεπτα διαβάζει τη θερμοκρασία και την υγρασία από έναν αισθητήρα DHT22.

    - Web Server: "Σερβίρει" την ιστοσελίδα στους χρήστες και δέχεται εντολές μέσω ενός API (π.χ. όταν πατάς το κουμπί στην ιστοσελίδα, γίνεται αποστολή ενός αιτήματος στο /api/toggle).

2. Η Διεπαφή Χρήστη (index.html & app.js)

    Αυτά τα αρχεία δημιουργούν την διεπαφή χρήστη, με την οποία μπορούμε να χειριστούμε τα LED μέσα από τον φυλλομετρητή:

    - Εμφάνιση: Μια σελίδα που δείχνει, σε πραγματκό χρόνο, τις μετρήσεις (Θερμοκρασία/Υγρασία) και έχει κουμπιά για τον έλεγχο των LED.

    - Διαδραστικότητα: "Ρωτάει" κάθε 5 δευτερόλεπτα τον μικροελεγκτή για τη νέα κατάσταση (fetch("/api/state")) και ανανεώνει τις μετρήσεις στην οθόνη.

3. Ρυθμίσεις (settings.json)

    Εδώ αποθηκεύονται οι παράμετροι λειτουργίας του ελεγκτή:

    - Τα στοιχεία του WiFi.
    - Τα Pins (επαφές) πάνω στις οποίες έχουν συνδεθεί η ταινία LED και ο αισθητήρας θερμοκρασίας/υγρασίας.
    - Το πλήθος των LED που διαθέτει η ταινία LED.
